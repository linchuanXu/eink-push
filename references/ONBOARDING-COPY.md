# 首次配置 / 首次推送成功后：功能引导

配置好后，以自然语言向用户展示以下内容。不要照抄，用对话语气呈现，挑 2–3 个最相关的例子重点说，其余可一笔带过。

---

配置好了！来看看你能用它做什么——

**📤 把内容推到墨水屏**

随时在对话结束后说「发到设备」，我会自动按字数选格式：

- 资讯卡 / 一屏摘要 → 单张卡片
- 多主题整理 / 简报 → 多张卡片集
- 长文 / 分析报告 / 书 → EPUB 电子书

也可以指定格式：
- "帮我搜一下 XX 最新进展，整理成简报发到阅星曈"
- "把这次对话的结论做成一张卡片推过去"
- "写一份 XX 深度报告，做成电子书发到设备"

---

**📚 查你的阅读数据**

我能直接读取你在阅星曈上的阅读记录：

- "我最近在读什么？" → 列出书架进度和阅读时长
- "帮我看看《XX》的书签" → 显示你划过的所有摘录

---

**✨ 把阅读数据变成内容推回去**

这是最好玩的部分——从你的书架和书签生成东西，再推回墨水屏：

- "把《XX》的书签做成精选卡片" → 引言排版的书摘卡片集
- "整理《XX》的阅读笔记，做成电子书" → 按章节归整的 EPUB
- "生成一张阅读看板" → 显示在读 / 已读 / 总时长的仪表盘卡片

---

**⏰ 定时推送**

- "每天早上 8 点推一张今日计划卡到阅星曈"
- "每周一早上整理本周计划，发到墨水屏"

---

随时说「发到阅星曈」「推一下」「我的书架」，我就会启动对应流程。
